# **App Name**: Control 7

## Core Features:

- Filter Bar: Date, ISO week, and product search filters.
- Production Table: A table for inputting planned weekly production and actual daily production, with calculations for daily and weekly compliance.
- KPI Dashboard: Displays key performance indicators (KPIs) such as total planned production, total actual production, variance, and percentage of completion.
- Weekly Summary: Summary of planned versus actual production, variance, and percentage of completion for each product.
- Data Persistence: Allows users to save the plan and daily production data.

## Style Guidelines:

- Primary color: Deep purple (#6750A4) for a sense of focused productivity.
- Background color: Light lavender (#F3EDF7) to ensure a gentle contrast with the primary color.
- Accent color: Soft blue (#B6CADB) for interactive elements and highlights, providing a calm but distinct contrast.
- Body and headline font: 'Inter', sans-serif, offering a modern, neutral, and machined appearance, suitable for both headlines and body text.
- Use a responsive layout to ensure the application is accessible and functional on various screen sizes.
- Align numerical inputs to the right for improved readability and data entry.
- Employ subtle animations, using Framer Motion, for a smooth user experience.
